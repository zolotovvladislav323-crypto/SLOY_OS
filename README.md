# SLOY OS

Защищённая меш-сеть без интернета.

SLOY OS — децентрализованная операционная система поверх Android/iOS/Windows/macOS/Linux, использующая все доступные радиочастотные и оптические каналы для создания невидимой, зашифрованной сети.

---

## Возможности

- Постквантовая криптография — Lattice-based + Double Ratchet
- Мультиканальная связь — WiFi Direct, Bluetooth LE, акустический модем, оптический канал (QR, стробоскоп, экран)
- Стелс-режим — маскировка трафика под HTTP/DNS/TLS, генератор ложного трафика
- Mesh-маршрутизация — DAG-топология, оптимизация пути, приоритеты пакетов
- Защищённое хранилище — зашифрованная NoSQL БД, VFS, автошреддер, Dead Man's Switch
- Форензика — спектр-анализатор, 3D-топология, мониторинг сигнала, лог пакетов
- Термоменеджмент — Duty Cycle, троттлинг при перегреве
- Биометрическая защита — FaceID / отпечаток, автоуничтожение при подозрительной активности

## Стек протоколов

| Уровень | Компоненты |
|---------|-----------|
| L1 Physical | DSSS-модулятор, поляризационный вращатель, детектор синхропоследовательности |
| L2 DataLink | Сборщик фреймов, избегание коллизий, фрагментация, приоритетный планировщик |
| L3 Network | Mesh-маршрутизатор, оптимизатор пути (Алгоритм Дейкстры) |
| L4 Transport | Гарантированная доставка с подтверждениями и ретраем |

## Сборка

```bash
# Установка зависимостей
dotnet workload install maui android

# Полная сборка
dotnet build

# Сборка Android APK
dotnet publish SLOY.App.Unified -f net8.0-android -c Release -o ./publish/android

# Сборка Windows
dotnet build SLOY.App.Unified -f net8.0-windows10.0.19041.0

# Сборка iOS (только на macOS)
dotnet build SLOY.App.Unified -f net8.0-ios -c Release

# Сборка macOS
dotnet build SLOY.App.Unified -f net8.0-maccatalyst -c Release
